目录组织方式

现在支持“模块 → 父目录 → 文章”的三级结构。您只要新建一个文件夹，并在其中放一个 index.md，它就能作为父目录页。

  • research/xxx/index.md：研究专题目录
  • translation/xxx/index.md：翻译专题目录
  • understanding/xxx/index.md：理解专题目录

示例专题

  • 个人研究 / 文档解析
  • 个人研究 / Agent 系统
  • 技术翻译 / LLM
  • 技术理解 / 结构化输出
