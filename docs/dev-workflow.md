# 论文阅读模板

这个页面可以作为你每篇论文的默认记录模板。

## 1. 基础信息

- 论文标题
- 作者 / 机构
- 发布会议或期刊
- 发布时间
- 论文链接 / PDF 地址

## 2. 快速摘要

- 这篇论文解决了什么问题
- 作者提出了什么核心方法
- 和已有方案相比，亮点在哪里
- 我为什么值得继续跟进这篇内容

## 3. 阅读拆解建议

| 阶段 | 建议动作 | 输出结果 |
| --- | --- | --- |
| 初读 | 先看摘要、引言、结论 | 判断是否值得精读 |
| 精读 | 拆方法、公式、图表 | 方法理解笔记 |
| 对比 | 找相似工作和差异点 | 横向比较结论 |
| 复盘 | 写个人启发和问题 | 可复用知识卡片 |

## 4. 常用记录模板

```md
# 论文信息

- 标题：
- 作者：
- 来源：
- 链接：

# 一句话摘要

# 核心问题

# 方法拆解

# 关键实验 / 图表

# 优点与局限

# 我的理解

# 可延伸阅读
```

## 5. 阅读后检查

- 是否写清楚论文要解决的问题
- 是否记录了核心方法和关键实验
- 是否补了自己的理解，而不只是摘抄
- 是否标注了后续可继续看的相关工作

## 6. 适合补充的内容

- 关键术语解释
- 公式推导过程
- 个人质疑点
- 可迁移到实际工作的启发
