# 文档目录

这里作为后续所有内容的总入口，建议按“长期有效的知识”和“专题整理”两类维护。

## 基础栏目

### 1. 论文与阅读

- [论文阅读模板]({{ '/docs/dev-workflow/' | relative_url }})
- 论文摘要、问题定义、方法拆解
- 关键图表、实验结果、个人启发

### 2. 技术与实践

- [技术文档模板]({{ '/docs/feature-examples/' | relative_url }})
- 框架学习、源码理解、接口约定
- 可复用工具函数、配置模板与代码片段

### 3. AI 与自动化

- Prompt 模板
- Agent 使用记录
- 自动化脚本、知识整理流程和工作流编排

### 4. 个人知识专题

- 前端专题
- AI 应用专题
- 论文方向地图与长期跟踪
