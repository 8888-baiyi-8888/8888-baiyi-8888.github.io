关于这个 GitHub Pages 项目

这是一个无需构建工具、可直接部署到 GitHub Pages 的静态站点项目。

适合放什么内容

  • 个人研究：系统设计、实验记录、论文方案、架构草图。
  • 技术翻译：外文博客、论文、官方文档、术语对照。
  • 技术理解：原理拆解、个人总结、学习笔记、实践心得。

图片如何存放

建议将图片放到 assets/images/ 目录下。示例图已放在 assets/images/samples/。

如何新增文章

  1. 在对应目录新增一个 HTML 文件，例如 research/new-article.html。
  2. 按照现有文章页面结构填写标题、日期、正文和图片。
  3. 在对应模块页面中增加一个文章卡片链接。
